## Introduzione

Il plugin **Oasis** permette di personalizzare gli oggetti nel gioco, aggiungendo effetti, modificando proprietà e molto altro. Questa funzionalità è particolarmente utile per i server che desiderano offrire un'esperienza di gioco unica ai propri giocatori.

## Comandi Principali

- **/itemcustom create <nome>**: Crea un nuovo oggetto personalizzato.
- **/itemcustom edit <nome>**: Modifica un oggetto personalizzato esistente.
- **/itemcustom delete <nome>**: Elimina un oggetto personalizzato.
