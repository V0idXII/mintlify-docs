**ItemCustom** è un plugin per il server Oasis che permette di creare oggetti personalizzati con attributi unici. Questo plugin è utilizzato per arricchire l'esperienza di gioco, permettendo ai giocatori di ottenere oggetti con caratteristiche speciali che non sono disponibili nei normali oggetti di Minecraft.

## Funzionalità

- **Creazione di oggetti personalizzati**: Gli amministratori possono creare oggetti con attributi personalizzati, come danni aggiuntivi, resistenza, effetti speciali, ecc.
- **Compatibilità con altri plugin**: ItemCustom è progettato per funzionare senza problemi con altri plugin del server Oasis.
- **Comandi personalizzati**: I giocatori possono utilizzare comandi specifici per ottenere o gestire gli oggetti personalizzati.
- **Configurazione flessibile**: Il plugin offre una configurazione dettagliata per gli amministratori, permettendo di definire esattamente come gli oggetti personalizzati devono comportarsi.

## Comandi

- `/itemcustom give <giocatore> <oggetto>`: Fornisce a un giocatore un oggetto personalizzato specifico.
- `/itemcustom list`: Mostra una lista di tutti gli oggetti personalizzati disponibili.
- `/itemcustom reload`: Ricarica la configurazione del plugin senza bisogno di riavviare il server.

## Permessi

- `oasis.itemcustom.give`: Permette di utilizzare il comando `/itemcustom give`.
- `oasis.itemcustom.list`: Permette di utilizzare il comando `/itemcustom list`.
- `oasis.itemcustom.reload`: Permette di utilizzare il comando `/itemcustom reload`.

## Installazione

1. Scarica il file `.jar` del plugin dalla pagina ufficiale di Oasis.
2. Inserisci il file nella cartella `plugins` del tuo server.
3. Riavvia il server per caricare il plugin.
4. Configura il plugin modificando il file `config.yml` nella cartella `ItemCustom`.
